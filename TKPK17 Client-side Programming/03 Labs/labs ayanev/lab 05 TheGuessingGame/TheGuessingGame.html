The Guessing Game

Description

Single player Guessing game. One turn in the game is played like this:

  1. The user guesses a number between 1 and 10.
  2. A random number is generated by the computer.
  3. If the user’s guess is lower than or equal to the random number, the user’s score is increased by the user’s guess.

When 10 turns have been played, the game is over. To make the game appear a bit more fun, the random number of balls is generated are drawn as yellow circles.
